## 구현 시 어려웠던 점
- 아직 C#이 처음이라 속성 창이나 도구 상자 등을 활용하는 게 쉽지 않았지만 과제와 실습 도중 여러 기능들을 탐색, 활용하며 조금씩 발전하고 있는 것을 느꼈다.
- GitHub주소를 버튼하고 연결할 때 오류가 발생해서 검색을 해 봤더니 UseShellExecute의 값을 true로 설정해 줘야 한다는 걸 알게 되었다.
- 사진 변경 버튼을 만들 때 이미지의 이름이 as로 C#예약어랑 겹친다는 사실을 몰랐어서 매우 헤맸었다. 검색을 통해 문제점을 알게 되어 이미지의 이름을 as_img로 수정했더니 해결되었다. 
- 사진의 경계선 스타일 변경기능을 만들 때는 다른 기능들과 달리 마우스 커서를 올리면 작동하는 기능이여서 private void button2_Click(object sender, EventArgs e)에서 button_Click이 부분만 바꾸면 된다고 생각하였다.
- 그래서 먼저 button_click 이 부분이 뭔지 찾아보았고 이벤트 핸들러라는 것을 알게 되었다. 그 후 MouseEnter를 활용해서 코드를 완성시켰고 잘 작동했다. (경계선 스타일 변경 명령어는 외부의 도움을 좀 받았다.)
